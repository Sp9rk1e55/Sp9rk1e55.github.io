A:long comparsion

  Monocarp wrote down two numbers on a whiteboard. Both numbers follow a specific format: a positive integer x with p zeros appended to its end.

Now Monocarp asks you to compare these two numbers. Can you help him?

Input
The first line contains a single integer t (1 $\leq$ t $\leq$ $10^4$) — the number of testcases.

The first line of each testcase contains two integers x1 and p1 (1 $\leq$ x1 $\leq$ $10^6$;1 $\leq$ p1 $\leq$ $10^6$) — the description of the first number.

The second line of each testcase contains two integers x2 and p2 (1 $\leq$ x2 $\leq$ $10^6$;1 $\leq$ p2 $\leq$ $10^6$) — the description of the second.

input:

using namespace std;

ll a[N];
void solve()
{
ll x1,p1,x2,p2;
cin>>x1>>p1>>x2>>p2;
ll c1=0,c2=0,d1=x1,d2=x2;
while(d1)c1++,d1/=10;
while(d2)c2++,d2/=10;
if(c1+p1!=c2+p2){
cout<<(c1+p1<c2+p2?”<”:”>”)<<endl;
}
else{
while(p1!=p2)
if(p1<p2) x2*=10,–p2;
else x1*=10,–p1;
if(x1<x2) cout<<”<”<<endl;
else if(x1>x2) cout<<”>”<<endl;
else cout<<”=”<<endl;
}
}
int main()
{
int t;

while(t–)
{

}
